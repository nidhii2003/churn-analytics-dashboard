# Enterprise Churn & Revenue Analytics Dashboard

An interactive operations dashboard designed to analyze subscriber behavioral patterns, calculate revenue risk, and identify high-value customer churn cohorts.

## 🚀 Live Demo
👉 [View the Active Dashboard] (https://nidhii2003.github.io/churn-analytics-dashboard/)

## 🛠️ Core Tech Stack & Concepts
- **Frontend UI Architecture:** Tailwind CSS
- **Dynamic Render Processing:** Modern Native JavaScript (ES6+)
- **Data Visualization Vector Layers:** Chart.js Integration
- **Analytical Competencies:** Cohort Segmentation, Risk Assessment Modeling, Financial KPI Metric Ingestion.
